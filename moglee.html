MVP? Existing Design System!

Given enough complexity, user interface can take a lot of time to build but when you’re a startup with limited budget, you might not be able to afford the cost. If the startup wants to save front-end costs and UI costs, then existing design systems are a great option. They do compromise on uniqueness to some extent but for testing an MVP, they are indeed perfect! For this reason, I picked Segment’s react based design system i.e Evergreen

Its a trust worthy design system which is supporting a massive data centric product which was recently acquired by Twilio for $3.2 billion.
